# FleetAI — Arabic-first SaaS for Car Rental Fleet Monitoring

FleetAI is an Arabic-first SaaS platform designed for car rental fleet monitoring in Saudi Arabia and the GCC.

## Roadmap

- [x] Project scaffold
- [x] Authentication (Supabase Auth)
- [x] Fleet Dashboard
- [x] Vehicle detail page
- [x] Alert system
- [x] Arabic / English language switch
- [ ] OBD data ingestion pipeline (In Progress)

*Note: Database schema, RLS, and real Auth are live.*

## Tech Stack
- Frontend: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- UI: Shadcn/ui + Cairo font (Arabic) + Inter font (English)
- i18n: next-intl
- Database & Auth: PostgreSQL via Supabase
- Charts: Recharts
- Payments: Moyasar
- Notifications: WhatsApp Business API + Twilio SMS
